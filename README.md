# 🚖 OLA Data Analysis Project

## 📌 Project Overview

This project analyzes OLA ride booking data using **SQL** and **Power BI** to generate meaningful business insights. The project covers the complete data analytics workflow, including data cleaning, SQL analysis, and interactive dashboard development.

---

## 🎯 Objective

The objective of this project is to:
- Clean and prepare raw booking data.
- Analyze ride booking patterns using SQL.
- Create an interactive Power BI dashboard.
- Extract business insights to support data-driven decisions.

---

## 📂 Dataset

- OLA Ride Booking Dataset
- Total Records: 100,000+
- Format: Microsoft Excel (.xlsx)

---

## 🛠️ Tools & Technologies

- SQL (MySQL)
- Power BI
- Microsoft Excel

---

## 🧹 Data Cleaning

The dataset was cleaned before analysis by:

- Removing duplicate records
- Handling missing values
- Correcting inconsistent data
- Formatting date and numeric fields
- Improving overall data quality

---

## 💻 SQL Analysis

Business questions answered using SQL include:

- Total Bookings
- Successful Rides
- Cancelled Rides
- Revenue Analysis
- Average Ride Distance
- Vehicle-wise Performance
- Payment Method Analysis
- Booking Status Analysis
- Customer Insights
- Driver Performance

---

## 📊 Power BI Dashboard

The interactive dashboard includes:

- KPI Cards
- Total Bookings
- Total Revenue
- Booking Trends
- Vehicle Type Analysis
- Ride Status
- Payment Method Distribution
- Dynamic Filters & Slicers

---

## 📈 Key Insights

- Identified booking trends over time.
- Compared vehicle performance.
- Analyzed successful and cancelled rides.
- Evaluated payment preferences.
- Generated business insights through visualizations.
